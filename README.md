# Habit Tracker

A simple Python tool to track and analyze daily habits

## Features
- Log daily habits (date, exercise, water intake, sleep, mood)
- Analyze habit statistics and trends
- Date tracking

## Files
- `habit_tracker.py` - Add new daily habit entries
- `analyze_habits` - View statistics and trends
- `habits.csv` - Data storage

## How to Use
- python habit_tracker.py
- python analyze_habits.py

## Future Improvements

- Weekly/monthly summaries
