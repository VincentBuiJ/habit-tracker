# Habit Tracker

A simple Python tool to track and analyze daily habits

## Features
- Log daily habits (date, exercise, water intake, sleep, mood)
- Analyze habit statistics and trends
- Date tracking
- Visualize mood distribution

## Files
- `habit_tracker.py` - Add new daily habit entries
- `analyze_habits.py` - View statistics and trends
- `visualize_habits.py` - Create charts from your data
- `habits.csv` - Data storage

## How to Use

**Track habits:**
```bash 
python habit_tracker.py
```

**View statistics:**
```bash 
python analyze_habits.py
```

## Visualize Your Data

View mood distribution as a bar chart:
```bash
python visualize_habits.py
```

## Requirements
```bash
pip install matplotbib
```
